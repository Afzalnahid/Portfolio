Project: E-Commerce Chatbot — Advanced Multimodal AutomationExecutive SummaryDesigned and deployed a high-performance E-Commerce Chatbot ecosystem for premium retail brands. This system automates the entire customer journey on Facebook Messenger—from visual product discovery and voice inquiries to real-time inventory checks and order fulfillment.  Technical Architecture1. Intelligent Input RoutingThe chatbot is built to handle the unpredictability of human communication through a sophisticated logic-based infrastructure:Media Processing: Uses a Switch node to instantly identify if a user has sent text, an image, a voice note, or a video.  Vision-Language Models: Integrated with Gemini 2.5 Pro and GPT-4o-mini to analyze product photos. The bot can identify specific jewelry motifs and finishes even from low-quality customer uploads.  Voice Commerce: Leverages Whisper-large-v3 to transcribe Bengali and Banglish audio, ensuring no customer is excluded based on their communication style.  2. The "Brain" (AI Orchestration)Autonomous Sales Agent: Powered by Gemini, the agent acts as a virtual floor manager, maintaining a professional persona and adhering to strict brand guidelines.  RAG System: Connected to a Supabase Vector Store to ensure every answer regarding price or stock is grounded in real-time data, eliminating AI hallucinations.  Smart Conversation Buffering: Utilizing a PostgreSQL backend, the bot "waits" for the user to finish a series of messages before responding, providing a single, coherent reply instead of several fragmented ones.  3. Data & OperationsAutomated Fulfillment: The AI is equipped with tools to write directly to Google Sheets, logging customer names, phone numbers, and delivery addresses automatically.  Professional Output Format: The bot is programmed to respond with a specific "Image + Text" sequence, providing a visually appealing product card followed by a price breakdown for every inquiry.  Tech StackEngine: n8n  Core AI: Google Gemini 2.5 Pro, OpenAI GPT-4o-mini  STT (Speech-to-Text): Whisper-large-v3 via Groq  Database & Memory: Supabase, PostgreSQL  Operational Tools: Google Sheets, Meta Messenger API  ImpactThis E-Commerce Chatbot transforms a standard messaging page into a fully autonomous 24/7 sales floor, capable of handling complex multilingual queries and manual-entry tasks with zero human intervention.

Project: Autonomous Social Media Content EngineCore Workflow & LogicThis system operates through two primary entry points: a time-based scheduler and a database trigger.  1. Content Generation PipelineData Analysis: The workflow starts by fetching recent post data (messages and reactions) from your Facebook page to understand current engagement.  AI Copywriting: An AI Agent, powered by Google Gemini, acts as an expert content creator. It is strictly instructed to write engaging 3-5 line captions in Bengali, complete with value-driven paragraphs and relevant hashtags.  Visual Creation: Simultaneously, the agent generates a high-quality 3D tech/business image prompt in English. This prompt is sent to an external AI image generator to create a custom 1080x1080 visual for every post.  Variety Engine: To prevent repetitive content, the system uses a random seed and strict logic to alternate between motivation, technical tips, and industry news while changing visual styles (e.g., Cyberpunk, Minimalist, or Professional Office).  2. Execution & DistributionMulti-Format Posting: A Switch node determines if the content is a Text or Image post. The system then uses the Facebook Graph API to publish the content directly to your page.  Media Hosting: Generated images are uploaded to ImgBB to create permanent URLs for documentation and secondary use.  Automated Scheduling: The Schedule Trigger is set to run three times daily (9 AM, 5 PM, and 11 PM) to ensure consistent page activity without manual input.  3. Logging & MonitoringCentralized Database: Every post—including the caption, media URL, status, and exact posting time—is appended to a Google Sheet (titled "Facebook Auto Post Content") for record-keeping.  Real-time Notifications: After every successful post or status update, the system sends a detailed summary to your Telegram manager bot so you can monitor operations on the go.  Technical HighlightsPrompt Engineering: Uses advanced system instructions to ensure AI output is strictly formatted as raw JSON for seamless data parsing.  Memory Integration: Includes a Simple Memory buffer to help the AI Agent maintain context and avoid repeating previously generated ideas.  Dynamic Image Generation: Integrates with image.pollinations.ai to generate free, high-quality, unwatermarked AI images on the fly based on the AI-generated descriptions.
